# EU Road Freight CO₂ Emissions Analysis

This repository contains an exploratory and econometric analysis of
road freight transport-related CO₂ emissions across European Union countries.

The main focus is on CO₂ emissions per ton-kilometer (CO₂ per tkm),
which allows for cross-country and temporal comparison of emission efficiency
in the road freight sector.

## Data
The analysis uses publicly available Eurostat data covering:
- EU member countries
- Time period: 2015–2022
- Road freight transport volume (ton-kilometers)
- Total CO₂ emissions from road freight transport
- Derived indicator: CO₂ per ton-kilometer (kg)

## Methodology
The following methods are applied:
- Descriptive statistical analysis
- Visualization of emission trends
- Ordinary Least Squares (OLS) regression
- Decision Tree regression to capture non-linear relationships

The analysis is implemented in Python using Jupyter/Google Colab.

## Repository Structure
